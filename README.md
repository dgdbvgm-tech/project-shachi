# Инвестиционное предложение | Собственный бренд одежды

## Архитектура проекта
* **Frontend:** Landing page (HTML5 / Tailwind CSS via CDN)
* **DevOps / CI/CD:** Интеграция автодеплоя через GitHub Actions и Firebase Hosting (пайплайн **DGD-SYNC**).

## Состав папок
* `/assets/images/` — Директория для хранения графики и изображений проекта.
* `index.html` — Главный файл лендинга с внедренными Open Graph мета-тегами.

*Разработано в соответствии со стандартом архитектуры **DGD Zero Entropy**.*
